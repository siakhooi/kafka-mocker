# kafka-mocker

Kafka Mock Messages Sender

## Installation

```
pip install kafka_mocker
```

## Usage

```
$ kafka-mocker -h
usage: kafka-mocker [-h] [-v]

Kafka Mock Messages Sender

options:
  -h, --help     show this help message and exit
  -v, --version  show program's version number and exit

```
